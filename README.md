# AI Society Game 🏙️

An interactive text-based society simulation game where multiple AI agents powered by **Grok from xAI** live, interact, trade, and build a society together.

## Features
- Realistic agent behaviors driven by Grok API
- Emergent social dynamics
- Resource management and daily decisions
- Easy to extend with more agents or rules

## Quick Start
1. Get your free xAI API key: [console.x.ai](https://console.x.ai)
2. `git clone https://github.com/ChrisBoardmanAsgard/ai-society-game.git`
3. `cd ai-society-game`
4. `pip install -r requirements.txt`
5. Set `XAI_API_KEY` environment variable
6. `python main.py`

Watch the society evolve over days!

Made possible by Grok.
